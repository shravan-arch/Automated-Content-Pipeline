# Limitations and Future Enhancements

## Limitations

- No OCR support for scanned PDFs
- Streamlit interface is single-user and local
- API rate limits apply under heavy usage
- Image understanding depends on textual descriptions

## Future Enhancements

- OCR integration for scanned documents
- Image-aware summarization
- Hosted deployment of Streamlit interface
- Keyword extraction and relevance scoring
- Multi-agent refinement pipeline
