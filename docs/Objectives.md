# Objectives and Scope

## Objectives

- To automate the summarization of large text documents
- To preserve relevance and clarity in generated summaries
- To support documents exceeding 10,000 words
- To provide both CLI-based and web-based interfaces
- To ensure quota-safe interaction with the Gemini API
- To demonstrate clean and modular software architecture

## Scope

The scope of the project includes text-based document summarization for academic, personal, and professional use. The system supports TXT, PDF, and DOCX formats and is designed for local execution on personal systems.

The project does not include OCR for scanned documents, real-time collaboration, or multi-user web deployment in its current version.
